Appointment System using Angular, PHP, REST API

This project is an Appointment System developed using Angular for the frontend, PHP for the backend, and a REST API for communication. It includes features like booking appointments, managing schedules, and user authentication. The system supports administrative, instructor, and student appointments and includes a queuing system to avoid lines.

Image Gallery with User Management:

This project features an interactive image gallery built with Angular for the frontend, PHP for the backend, and a REST API for seamless data handling. Users can browse thumbnails, view full-size images, and upload their own with basic validation. Additional functionalities include image editing, filters, and commenting, all designed with a focus on enhancing user experience and ease of use.
